This folder contains all the system code in a single ZIP file.

To run the system:

Download and extract the ZIP file into C:\xampp\htdocs.
Open the XAMPP Control Panel and start Apache and MySQL.
Ensure you have imported the SQL database into phpMyAdmin via localhost.

To set up the SQL database:

Open phpMyAdmin from localhost.
Create a database named uniscope_magazine.
Import the SQL file from this folder into the database.
Ensure the project files are already extracted into the htdocs folder.
